# Policy Timeline Report (房市政策時光機)

## 1. 概述
`PolicyTimelineReport` 是一個視覺化工具，將建案的銷售期間與台灣房市重大政策、金融事件進行時間軸對照。幫助用戶理解政策變動（如房地合一稅、升息）對特定建案銷售的影響。

## 2. 功能特點
- **互動式時間軸**: 
  - X軸為時間（年份），支援縮放 (`ZoomIn`/`ZoomOut`)。
  - **建案銷售區間**: 以橫條圖顯示所選建案的銷售起訖時間。
  - **政策事件點**: 在時間軸上標示政策（紅色）與金融事件（綠色）。
- **建案篩選**: 支援多選建案，將其銷售期疊加在時間軸上進行比較。
- **事件詳情面板**: 
  - 點擊時間軸上的事件，下方會展開詳細資訊。
  - 包含：政策內容、具體變革、時代背景、受眾影響分析。
  - 支援展開查看詳細條文。
- **政策比較表**: 列表展示所有選定時間範圍內的政策，支援匯出。

## 3. props 定義
- `data`: 交易列表陣列，用於提取建案名稱與計算銷售起訖時間。

## 4. 關鍵邏輯
- **區間計算 (`projectPeriods`)**: 
  - 遍歷交易數據，找出每個建案的最早與最晚交易日，定義出銷售生命週期。
- **座標定位 (`getPosition`)**: 
  - 將日期轉換為時間軸上的百分比位置 (0% - 100%)，以適應不同縮放級別。
- **事件過濾**: 
  - 根據當前視圖的年份範圍，篩選顯示 `REAL_ESTATE_POLICIES` 中的靜態政策數據。
- **視覺化細節**: 
  - 若政策發生在建案銷售期間內，該政策節點會自動放大並高亮，強調其相關性。
